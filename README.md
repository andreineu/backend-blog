# Backend

# Stack

- nodejs
- express
- apollo-server-express
- type-graphql
- typeorm
- dataloder

#### authentication: 
- express-session
- redis

# How to run this project:

1. Run `npm i` command
2. Setup env variables
3. Run `npm start` command

Env variables

- SESSION_SECRET
- DATABASE_URL
- REDIS_URL
- PORT
- CORS_ORIGIN
